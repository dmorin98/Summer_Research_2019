The COSMOS script editor allows you to create your program, test it, and save the program to a file when you have it running correcly.This option could also be done using any program that can create a .txt extension file. However using the COSMOS script editor, you can be assured that your code is formatted correctly and even automaticly add comments to the code.



Test the Code Button



The “Test the Code” button is used to send your code to the VXM as a test, and run that block of code.



Auto-Format Code



The “Auto-Format Code” button is used to reformat your code into a structure ends each command with a comma for readability,and to ensure the string is not mistakenly mis-understood by the VXM. This option does not account for spelling errors or commands that the VXM does not understand. (Example seen below)

Auto-Format and Comment Code



The “Auto-Format and Comment Code” does much like the “Auto-Format Code” button but it also adds comments for the commands in plain english for later readablity, and so that the VXM does not mis-understand a comment as a command.
