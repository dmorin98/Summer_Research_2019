COSMOS comes with several different troubleshooting tools. Each to assist in tracking down a problem should one arise. These tools include:

Troubleshooting User Inputs: This tool is used to continuously track the state of each input and show the user the current state of each



Troubleshooting Limits allows the user to see a graphical representation of the state of the limits. This tool can be especially helpful in situations that the axis is not operating correctly



Troubleshooting Multifunction User I-O tool is basicly a graphical representation of the diagnostic tool for Multifunction User I-O.



Troubleshooting User Outputs is a graphical representation of the user outputs as they toggle on and off. This tool may be the most powerful in the case that you have no other way of indicating that the outputs are operational. (such as LED's or a test kit)
