The administrator settings in COSMOS allows you to change the look of COSMOS somewhat to simplify the end users tasks. Certain menu options can be shown or hidden depending on the menu items hidden. If you have personnel that you do not want accessing parts of COSMOS, then use this option to hide the parts you do not wish to be seen. This menu option can be found under “Tools--->Administrator--->COSMOS Settings”



To enter the Administrator tools of COSMOS, you must first enter the password. If you do not wish end users to have this password then do not give them this page of the manual. (The password was intentionally set to be easy to guess however).



The COSMOS settings form allows for each of the options that can be changed within COSMOS. For readability sake, the layout is the same as the layout of the COSMOS menu bar.



The most powerful tool within COSMOS is the Driver Debug Window which can allow you to see exactly what is being sent to and from the VXM as it is seen by the driver itself.

Under most cases, it is advised to only use this tool to view the commands, as opening and closing the port and other available actions on this window may have unexpected results within COSMOS.
