The single axis quickmove allows the user to quickly move an axis a desired distance at a given rate of speed. This tool also allows for using your own units (such as inches, millimeters, revs, degrees, or any sort of units you wish). All you need to do is enter the distance per step multiplier and the units you wish to use. Then COSMOS will take this value into account when sending commands to the VXM for movement.
