To use the AutoReverse, do the following:

  1. go to “Tools--->Quick-Moves--->AutoReverse” and the form above will appear

  2. From the top down, you will see several sections to be filled in. These sections are as follows:

Section

Option

Description

Start




From Current Location

Starts the autoreverse from the current position


From Negative Limit

Starts from negative limit (if you enter 0 in the number of steps) or starts X number of steps from the negative limit. You can also select if you wish the homing routine run every time the program is run, or just run the 1st time after a power-cycle.


From Positive Limit

Starts from positive limit (if you enter 0 in the number of steps) or starts X number of steps from the positive limit. You can also select if you wish the homing routine run every time the program is run, or just run the 1st time after a power-cycle.


Speed

Enter the speed to travel at (1 to 6000 steps per second)


Analog Range

Select the analog speed range to travel at (See Chart Below)

As you may have noticed by now, as each section is filled in, COSMOS updates itself to show the VXM code to match your choices. Next all you need to do is one (or more) of the following:

Test the Code



Tests the code that you just created

Save to VXM



Saves the code to the VXM so it is retained after a power-cycle

Save to File



Saves a copy to a script file for future use



COSMOS does one final check to make sure it is matched up with the choices that you made, then sends it to the VXM or Saves it to a file if that was the option you have chosen.
