To set your attached devices do the following:

  1. go to “Tools--->Set Up Attached Devices” and then choose the Axis that is to be set up.

  2. Choose the device that is to be set for the axis

    Jog Button Speed

    To set your jog buttons, enter the desired speed and press the save button

    (Warning, changing this speed will also change the digital joystick speeds to match)



    Digital Joystick

    The “Set Digital Joystick” form will allow you to set the primary and secondary speeds of your joystick (speed selected by the button on the top of the joystick)

    To set your joystick, enter the desired speed and press the save button

    (Warning, changing this speed will also change the front panel jog button speeds to match)



    Analog Joystick

    The “Set Analog Joystick” form will allow you to set the primary and secondary speed ranges for the analog joystick

    To set your joystick, enter the desired speed and press the save button



    Thumbwheel Selector Switch

    To set the VXM to use the optional Thumbwheel Selector Switch, check the checkbox

    (Warning: This option is not to be used in conjuction with the Analog or Digital Joysticks)
