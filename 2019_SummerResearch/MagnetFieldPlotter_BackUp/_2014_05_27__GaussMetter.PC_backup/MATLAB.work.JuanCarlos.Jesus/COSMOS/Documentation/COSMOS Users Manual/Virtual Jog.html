To use the Virtual Jog, Do the following:

  1. go to “Tools-->QuickMoves--->Virtual Jog” (The form above will appear)

  2. Select the motor that you wish to jog

  3. Enter the Velocity that you wish to jog the motor

  4. Press the button that corresponds to the direction you wish to travel (if a limit is hit, the LED for that direction will light up)
